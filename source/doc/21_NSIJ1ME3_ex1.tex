\subsection*{Partie A}

\begin{enumerate}
\item Expliquer le rôle du protocole TCP et du protocole IP dans un réseau informatique.

\item On considère un ordinateur dont les paramètres réseau sont les suivants:
\begin{itemize}
\item Adresse IP: 200.100.10.60
\item Masque du sous-réseau : 255.255.255.0
\end{itemize}
\begin{enumerate}
\item Donner l'identifiant (adresse) du réseau.
\item Donner la première et la dernière adresse IP qui peuvent être affectées à un hôte.

En déduire le nombre de machines hôte identifiables sur un tel réseau.
\end{enumerate}
\end{enumerate}
