\section*{Couches réseau et protocoles}

La communication entre machines aujourd’hui se réalise grâce à différents éléments physiques et protocoles de communication. On les catégorise en 4 couches différentes.

\begin{enumerate}
\item La \textbf{couche physique} s'appuie sur des interfaces réseau qui sont identifiables de manière unique dans un réseau. Comment une interface réseau est-elle identifiée de façon unique ?

\item La couche internet permet la communication de 2 machines appartenant à 2 réseaux différents. Quel est le protocole utilisé et comment cela se traduit-il sur les machines ?

\item La couche transport assure l'acheminement des données et vérifie que celles-ci sont complètement arrivées.

\begin{itemize}

\item Citer 2 protocoles de la couche transport pour les échanges sur internet. 
\item Comment procèdent ces protocoles pour assurer la transmission des données ?

\end{itemize}

\item La couche application initie un échange de données sur internet. Citer des applications qui initient et échangent des données sur internet.

\item Les différentes couches traversées par un paquet de données à envoyer sur internet sont complétées par chacune des couches : c'est l'encapsulation des données. Représenter par un schéma cette encapsulation.

\end{enumerate}

On va étudier un réseau avec le logiciel filius qui permet simuler des réseaux. Le réseau ci-dessous est à télécharger sur le moodle de l'ENT.

\item Le réseau L2 comprend 1 serveur qui a pour adresse IP 192.168.6.10

\begin{enumerate}
\item Comment s'appelle les appareils qui relient les réseaux L1 et L2 ?

\item Combien d'interfaces réseau contient le routeur R3 ? Donner l'adresse réseau de chacune des interfaces.

\item Compléter le schéma de réseau en indiquant toutes les adresses réseaux de chaque machine.

\item La commande \textbf{ping} \textit{ip de la machine distante} permet de vérifier la connexion avec une autre machine du réseau. Tester cette commande entre le portable et le serveur.

\item La commande \textbf{traceroute} \textit{ip de la machine distante} affiche le parcours entre deux machines et affiche les routeurs utilisés. Combien de routeurs sont traversés dans cette communication ? Lesquels ?

\end{enumerate}

\section*{Table de routage}

Les routeurs sont des appareils munis de plusieurs \textbf{interfaces} réseaux qui permettent aux routeurs de communiquer entre eux. Les interfaces sont désignées par leur nom ou par leur adresse IP. \medskip

Une \textbf{passerelle} est un routeur ou une interface du routeur qui permet d'accéder au réseau de destination.\medskip

Sue la figure ci-dessous, les réseaux L1 et L2 sont reliés par différents routeurs.
