Exercices#

Exercice 1#

  1. Retrouver sur votre machine l’adresse physique de l’interface réseau utilisée.

  2. Quelle est l’adresse IP de votre machine ?

  3. Quel est le masque de sous réseau ? Que signifie cette valeur ?

  4. En déduire l’adresse du réseau.

  5. Retrouver la passerelle utilisée par votre ordinateur.

Exercice 2#

Une machine dans un réseau local a pour adresse IP 192.168.1.34/23.

  1. Donner le masque de sous-réseau en notation décimale.

  2. En déduire l’adresse de réseau auquel appartient cette machine.

  3. Combien de machines peuvent se connecter au réseau ?

  4. La passerelle est la dernière adresse IP de la plage d’adresse possible. Quelle est la passerelle indiquée sur cette machine ?
